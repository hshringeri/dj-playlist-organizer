# DJ Playlist Organizer

Spotify-based DJ library organizer with intelligent folder classification.

## Features

- Spotify OAuth integration
- Listening history sync & analysis
- DJ folder classification (18 categories based on set position, texture, rhythm, emotion)
- Smart playlist generation
- BPM and key detection

## Setup

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and add your API keys
4. Run: `python src/sync.py`
